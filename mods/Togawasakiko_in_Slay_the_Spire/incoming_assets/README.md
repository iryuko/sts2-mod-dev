# 来稿投递区

本目录只放原始来稿，不直接当作最终安装资源使用。

当前规则：

- 你把素材先投到对应子目录
- 文件名可以先用直观描述，不要求一开始就对齐最终正式名
- 我会再按原版规格整理到 `assets/`

- `cards/normal_pool/`
- `cards/pressure_generated/`
- `ancients/event_main/`
- `ancients/map_node/`

补充说明：

- `ancients/` 这一组用于“原创先古之民”来稿，不放到工作区根下的参考导出目录里处理。
- `ancients/event_main/`
  - 先古之民事件主立绘
  - 建议 `2560x1200 PNG`
- `ancients/map_node/`
  - 地图节点头像
  - 建议 `208x208 PNG`
- `ancients/map_node_outline/`
  - 地图节点头像 outline
  - 建议 `208x208 PNG`
- `ancients/dialogue_icon/`
  - 事件对话头像
  - 建议 `88x88 PNG`
- `ancients/dialogue_icon_outline/`
  - 事件对话头像 outline
  - 建议 `88x88 PNG`
- `audio/music/full_tracks/`
  - 完整音乐来稿
  - 建议优先交 `OGG`
- `audio/music/song_cues/`
  - 歌曲牌短版或切片 cue
  - 只有完整曲目不适合直接播放时才单开
- `audio/sfx/character_select/`
  - 角色选择界面的独特音效
  - 建议短音效交 `WAV` 或 `OGG`
- `audio/sfx/jukebox/`
  - 点歌系统自身的打开、确认、切歌、停止等音效
- `audio/sfx/ui/`
  - 角色局内功能相关的其它通用短音效
