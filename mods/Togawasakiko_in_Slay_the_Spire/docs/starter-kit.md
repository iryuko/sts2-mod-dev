# 初始 Relic 与起始牌说明

日期：2026-03-24

## 一 本文件的定位

本文件记录的是角色牌系中的起步子集，不是另一套独立体系。

当前应这样理解：

- 起始 relic 是角色首局节奏的点火入口。
- 起始牌是角色整体牌组设计中的 `Basic` 子集。
- 它们与后续普通 / 非凡 / 稀有牌共享同一套机制主轴，而不是额外拆出一套“起始牌专用规则”。

## 二 初始 relic

当前已冻结的显示层方案：

- 显示名：【人偶的假面】
- 效果：每个玩家回合开始时，给予所有敌人 `1` 层压力。

当前定位：

- 这是一个纯点火 relic。
- 作用是稳定开启压力系统，而不是直接承担高额价值。
- relic 文本只轻度提示压力体系，不应塞入过多规则映射。

当前冻结候选英文内部名：

- `DollMask -> DOLL_MASK`

当前记录原则：

- 先按这个名称写入项目文档并继续使用。
- 显示名仍为【人偶的假面】。
- 若后续统一命名审校需要再改，应统一处理，而不是在设计文档内零散改名。

## 三 起始牌组

当前已确认：

- `4` 张打击
- `4` 张防御
- `1` 张自定义白色攻击牌
- `1` 张自定义白色技能牌

说明：

- 基础打击 / 防御的数值与原版无区别。
- 但应使用角色专属卡图版本。
- 这 `10` 张起始牌应视为角色整体牌系中的起步配置，而不是独立于正常设计体系之外的特殊集合。

## 四 两张专属起始牌

### 1. 起始攻击牌

- 当前冻结候选显示名：
  - 中伤 / `Slander`
- 费用：`1`
- 基础伤害：`4`
- 追加规则：根据目标身上的压力层数增加伤害
- 当前已确认倍率：每层 `+3`

升级后：

- 费用改为 `0`
- 基础伤害仍为 `4`
- 每层压力 `+3` 不变

本地核查：

- 已在本地原版资源、本地化表与命名研究资料中检索 `Slander`
- 当前未发现明显原版命名冲突
- 因此 `Slander` 当前继续保留，作为本项目中的冻结候选名使用

### 2. 起始技能牌

- 当前冻结候选显示名：
  - 难熬 / `Unendurable`
- 费用：`2`
- 基础效果：获得 `5` 格挡
- 追加效果：令 `1` 个目标获得 `2` 层压力

升级后：

- 费用仍为 `2`
- 获得 `8` 格挡
- 令 `1` 个目标获得 `2` 层压力

本地核查：

- 已在本地原版资源、本地化表与命名研究资料中检索 `Unendurable`
- 当前未发现明显原版命名冲突
- 因此 `Unendurable` 当前继续保留，作为本项目中的冻结候选名使用

## 五 它们在整体牌系中的作用

这套起手组件表达的不是直接爆发，而是：

- 先建立压力
- 再把压力转为更高效率的攻击或衍生资源

因此这些起始组件共同承担：

- 教会玩家认识“目标身上的压力”很重要
- 让角色从首战开始就具备最小压力闭环
- 为后续普通 / 非凡 / 稀有牌的设计预留统一主轴

后续设计要求：

- 不把起始牌单独写成另一套效果哲学。
- 起始牌的升级与定位，应直接放回整套角色卡系里比较。
- 后续普通 / 非凡 / 稀有牌应该是在这套起步逻辑上展开，而不是与其割裂。
