# 角色机制总览

日期：2026-03-24

## 角色定位

`Togawasakiko_in_Slay_the_Spire` 当前已确认的核心设计方向：

- 通过 debuff 控制敌人。
- 通过生成压力衍生牌打出连锁节奏。
- 主轴不是纯爆发角色，而是先建立控制与压制，再兑现收益。

## 战斗身份优先级

- 控制
- 压制
- 防御
- 处决

说明：

- 处决类效果允许存在。
- 但处决不是主框架。
- 处决更适合作为高稀有度 payoff，而不是整套角色的基础运行方式。

## 当前主机制主轴

- 主资源主题是“压力”。
- 压力本身不是自动结算伤害或自动触发收益的状态。
- 压力的意义在于被牌效与 debuff 消耗，并兑换为压力衍生牌。
- 当前主线优先围绕“敌方压力”展开，而不是围绕“我方压力”先做对称系统。
- 当前冻结工作版的压力衍生牌数量为 `4` 张。

## 角色牌系的一体化原则

当前应把这名角色的所有非衍生牌视为同一套角色牌系中的不同子集，而不是拆成多套体系。

这包括：

- 基础打击 / 防御
- 两张专属起始牌
- 后续普通 / 非凡 / 稀有卡

当前原则：

- 起始牌不是独立系统。
- 起始牌只是整套角色牌系里的 `Basic` 子集与起步教学入口。
- 它们与后续卡牌一样，都应放在同一设计主轴下考虑：
  - 效果定位
  - 升级路线
  - 与压力体系的关系
  - 与 debuff 体系的关系

## 当前已冻结的对象边界

- 核心状态：
  - 压力 `Pressure`
- 当前冻结工作版压力衍生牌：
  - 人格解离
  - 自闭
  - 满脑子都想着自己
  - 过劳焦虑
- 原创 debuff：
  - 人格解离 `Persona Dissociation`
  - 自闭 `Social Withdrawal`
  - 自卑 `Inferiority`

说明：

- 压力是核心状态，不是普通 debuff。
- 人格解离、自闭、自卑属于原创 debuff 层。
- 人格解离当前正式规则为：
  - 可叠层
  - 每层都会让下一次受到的伤害在格挡结算前先翻倍
  - 按每次独立伤害事件逐层结算
  - 触发后只失去 `1` 层
  - 若这次伤害事件最终被格挡到 `0`，该层仍会照常消耗
  - 若一直没有受伤，则不会自行消失
- 压力衍生牌是独立的一类牌，不属于正常卡池。
- 起始牌属于同一套角色牌系中的 `Basic` 子集，不因起始身份而脱离统一设计体系。

## 当前已实现的风险换收益样例

- `Treasure Pleasure` 已作为 `song` 非凡技能牌接入正常卡池。
- 它的实现口径是：
  - 自己获得 `1` 层人格解离
  - 同时获得临时 power【磁场力量-地狱战神】
  - 该 power 持续到本回合结束
  - 在持续期间内，你打出的每一张攻击牌都会额外结算 `1` 次
