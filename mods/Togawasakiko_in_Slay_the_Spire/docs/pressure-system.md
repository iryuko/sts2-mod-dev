# 压力系统说明

日期：2026-03-24

## 一 压力的定位

- 压力是本角色的核心状态层。
- 压力与普通 debuff 分层记录，不混为同一类对象。
- 当前主线优先围绕敌方压力展开。

## 二 压力的基础规则

当前已确认：

- 敌我双方都可能拥有压力。
- 敌人与我方的交互逻辑不同。
- 压力不衰减。
- 压力无上限。
- 压力没有基础效果。
- 压力不会自己结算收益。
- 压力只会被各种牌效与 debuff 消耗，然后兑换压力衍生牌。
- 单体敌人独立计算，不做全场共享。

## 三 当前推荐的系统理解

现阶段应把压力理解成：

- 一个可累积的战斗标记层
- 一个被动储备资源
- 一个为“控制 -> 压制 -> 连锁衍生牌”服务的中间媒介

不应把压力直接写成：

- 自动伤害
- 自动易伤类倍率
- 自动回合衰减状态
- 全场统一共享槽

## 四 敌方与我方压力的当前边界

当前已确认：

- 敌我双方都可以持有压力。
- 但两侧交互逻辑不同。
- 当前主机制优先围绕敌方压力建立。

当前仍未冻结：

- 我方压力在首版中的实际参与度
- 我方压力是否在首发卡池中拥有稳定消费路径
- 敌我双方压力是否会共享同一套衍生牌触发语义

## 五 压力衍生牌统一规则

当前已确认：

- 当前冻结工作版的压力衍生牌总数为 `4` 张。
- 压力衍生牌全部生成到手牌。
- 压力衍生牌全部默认虚无。
- 压力衍生牌全部默认消耗。
- 压力衍生牌可以被正常减费、复制、升级。
- 压力衍生牌不进入正常卡池。
- 压力衍生牌可以通过控制台调用。

补充说明：

- 当前不考虑通过 relic 直接生成压力衍生牌。

## 六 当前冻结工作版压力衍生牌

说明：

- 以下 `4` 张牌均继承第五节的统一规则，默认生成到手牌、默认虚无、默认消耗。

### 1. 人格解离

触发来源：

- 由“虚弱”消耗压力兑换

冻结规则：

- 消耗 `2` 层压力
- 获得技能牌【人格解离】
- 费用：`1`
- 默认消耗
- 效果：对指定怪物添加 debuff【人格解离】

对应 debuff：

- 【人格解离】效果：
  - 本次生成牌默认施加 `1` 层
  - 每层都会让下一次受到的伤害在格挡结算前先翻倍
  - 按每次独立伤害事件逐层结算
  - 每次触发后失去 `1` 层
  - 即使最终被格挡至 `0`，该层也会照常消耗
  - 不会因为回合结束自动消失

### 2. 自闭

触发来源：

- 由“失去力量 / 失去敏捷”消耗压力兑换

冻结规则：

- 消耗 `1` 层压力
- 获得技能牌【自闭】
- 费用：`0`
- 默认消耗
- 效果：使指定怪物获得 `3` 层【自闭】

对应 debuff：

- 计数型持续 debuff
- 每次在回合结束造成 `7` 点伤害后减少 `1` 层
- 可叠加
- 可续层

### 3. 满脑子都想着自己

触发来源：

- 由“易伤”消耗压力兑换

冻结规则：

- 消耗 `3` 层压力
- 获得攻击牌【满脑子都想着自己】
- 费用：`3`
- 默认虚无
- 默认消耗
- 造成 `9` 点伤害
- 晕一回合

命名冻结：

- 英文显示名：`All You Think About Is Yourself`
- 内部类名：`AllYouThinkAboutIsYourself`
- `Entry`：`ALL_YOU_THINK_ABOUT_IS_YOURSELF`
- 资源文件名：`all_you_think_about_is_yourself.png`

说明：

- 这张牌的伤害部分是为了配合“易伤”前提，不是单纯只有控制效果。

### 4. 过劳焦虑

触发来源：

- 由【自卑】消耗 `1` 层压力兑换

冻结规则：

- 名称：【过劳焦虑】
- 类型：技能牌
- 费用：`0`
- 默认消耗
- 效果：抽 `1` 张牌

说明：

- 这张牌属于压力衍生牌，不进入正常卡池。
- 当前先保持效果简洁，不在本轮继续扩写。
- 本轮已冻结它与【自卑】debuff 的对应关系。

## 七 当前设计含义

这意味着压力衍生牌更接近：

- 角色私有的战斗内兑换物
- 由压力消费行为触发的临时工具牌

而不是：

- 正常奖励池卡
- 起始牌组常驻卡
- relic 常规产出物

## 八 当前仍未冻结的细项

本文件不在本轮写死以下内容：

- 各衍生牌在后续升级后是否会出现个别“非虚无”特例
- 具体消费比例在完整卡池中的铺开方式
- 敌我双方各自能兑换哪些衍生牌
- 各衍生牌在完整卡池里的分布密度
