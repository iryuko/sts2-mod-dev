## 二 本批次统一规则

### 1. `song` tag 定位

- `song` 是普通逻辑 tag。
- `song` 不自带任何默认规则。
- 所有收益都必须由具体卡牌或 power 明确写出。

### 2. 歌曲牌池规则

- 歌曲牌池包含所有带 `song tag` 的正常卡池牌。
- 歌曲牌池不包含压力衍生牌。
- 歌曲牌池不包含单纯为生成而存在的 `Token` 牌。

### 3. 压力衍生牌池规则

当前固定为 `4` 张：

- `Persona Dissociation`
- `Social Withdrawal`
- `All You Think About Is Yourself`
- `Overwork Anxiety`

当前要求：

- 压力衍生牌不进入正常卡池。
- 只能通过既定机制生成。
- 未明确追加前，不要自行扩池。

### 4. 正常卡池与商店规则

- 歌曲牌可正常出现在战斗奖励中。
- 歌曲牌可正常出现在商店中。
- 压力衍生牌不进入正常奖励池与商店池。

### 5. 资源与实现边界

- 本批次如缺正式卡图或 icon，可继续使用占位资源。
- 但文档必须明确标记：
  - 已接入且文件存在
  - 已接入但仍是占位
  - 代码已引用但资源缺失

## 四 单卡冻结规格

### 1. `Compose / 谱曲`

- 类型：技能牌
- 稀有度：非凡
- 费用：`1`
- 升级后费用：`0`
- `song tag`：否
- 基础效果：
  - 随机将 `1` 张歌曲牌加入手牌
  - 该牌本场战斗费用变为 `0`
  - `Compose` 本身消耗
- 对象来源：
  - 必须从“歌曲牌池”中随机生成
  - 不是从全卡池随机

### 2. `Ave Mujica / Ave Mujica`

- 类型：能力牌
- 稀有度：稀有
- 费用：`3`
- 升级后费用：`2`
- `song tag`：是
- 基础效果：
  - 每回合开始时，在常规抽牌之后，随机将 `1` 张压力衍生牌加入手牌
  - 然后自动打出抽牌堆顶的第 `1` 张牌
- 额外规则：
  - 如果抽牌堆为空，则跳过“自动打出抽牌堆顶第 `1` 张牌”
  - 生成对象必须来自“压力衍生牌池”

### 3. `Ether / 以太`

- 类型：攻击牌
- 稀有度：非凡
- 费用：`1`
- `song tag`：是
- 基础效果：
  - 造成 `3` 点伤害 `3` 次
- 追加效果：
  - 若本牌击杀目标，使所有其他敌人获得 `1` 层压力
- 升级后：
  - 若本牌击杀目标，使所有其他敌人获得 `2` 层压力
- 升级说明：
  - 伤害本身不升级
  - 只升级击杀后的压力层数

### 4. `Crucifix X / 十字架X`

- 类型：攻击牌
- 稀有度：非凡
- 费用：`X`
- `song tag`：是
- 基础效果：
  - 造成 `6` 点伤害 `X` 次
- 条件追加：
  - 若场上所有敌人都至少拥有 `4` 层压力，则额外造成 `2` 次伤害
- 升级后：
  - 每次伤害改为 `7`
- 额外规则：
  - 即使 `X = 0`，只要满足“所有敌人至少 `4` 层压力”，仍然会打 `2` 次
- 结算说明：
  - 实际攻击次数为“`X` 次基础攻击 + `2` 次额外攻击”
  - runtime 实现已改为命令层的“全体对手 + 多段数”写法，接近原版 `Whirlwind`
  - 不再使用“卡面 `AllEnemies` + 手写逐敌人 `Targeting(enemy)`”的旧实现，以避免与 `Corrupted` 附魔的目标处理冲突

### 5. `Face / 颜`

- 类型：技能牌
- 稀有度：普通
- 费用：`1`
- `song tag`：是
- 基础效果：
  - 获得 `8` 点格挡
  - 获得一个持续到“你的下个回合开始前”的临时 Power
- 临时 Power 效果：
  - 在此期间，每当你受到来自怪物的伤害时，对伤害来源施加 `3` 层压力
- 升级后：
  - 格挡变为 `12`
- 实现说明：
  - 按“一回合持续型 Power”处理
  - 参考原版临时反击 / 防反类效果

### 6. `Saki, Move Plz / 祥，移动`

- 类型：攻击牌
- 稀有度：普通
- 费用：`1`
- `song tag`：否
- 基础效果：
  - 造成 `9` 点伤害
- 条件追加：
  - 若你本回合打出的上一张牌是歌曲牌，使目标获得 `1` 层易伤
- 升级后：
  - 伤害变为 `13`
- 额外规则：
  - 如果这是本回合打出的第一张牌，则默认不触发易伤

### 7. `Music of the Celestial Sphere / 天穹之乐`

- 类型：技能牌
- 稀有度：非凡
- 费用：`2`
- `song tag`：是
- 目标：所有敌人
- 基础效果：
  - 使所有敌人每有 `5` 层压力便失去 `1` 点力量
- 升级后：
  - 改为每有 `4` 层压力便失去 `1` 点力量
- 结算说明：
  - 按向下取整处理

### 8. `KillKiss / KillKiss`

- 类型：能力牌
- 稀有度：稀有
- 费用：`1`
- `song tag`：是
- 基础效果：
  - 怪物回合开始时，若其压力层数严格大于其当前剩余生命值的一半，则对其造成 `25` 点伤害
- 升级后：
  - 改为造成等同于其最大生命值的伤害
- 额外规则：
  - 判断条件使用“严格大于”，不是“大于等于”

## 五 实现优先级建议

建议按以下顺序实现：

1. 先补 `song` tag 与两类固定池工具函数
2. 再补 `Compose` 与 `Ave Mujica` 的池生成逻辑
3. 再补 `Ether`、`Crucifix X`、`Saki, Move Plz` 的条件判定
4. 再补 `Face` 与 `KillKiss` 的事件型 power
5. 最后补 `Music of the Celestial Sphere` 的群体压力换算
